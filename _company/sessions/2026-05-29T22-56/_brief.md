# 📋 작업 브리프

**원 명령:** 아니야 아니야. 모든 업무 지금 당장 중지해.

## 요약
사용자의 명시적인 명령에 따라 모든 현재 진행 중인 업무와 작업 흐름을 즉시 중단합니다. 이는 비상 정지(Emergency Halt) 상태를 의미하며, 추가적인 작업 지시가 있을 때까지 모든 에이전트의 작업을 보류합니다.

## 분배
- **📱 영숙**: 모든 현재 진행 중인 작업을 '비상 정지(Emergency Halt)' 상태로 선언하고, 모든 에이전트(youtube, instagram, designer, developer, business, writer, researcher)에게 작업을 중지할 것을 공식적으로 통보합니다. 이후, 사용자에게 다음 지시가 있을 때까지 대기 상태를 유지하고, 시스템의 안정성을 최우선으로 보고합니다.
